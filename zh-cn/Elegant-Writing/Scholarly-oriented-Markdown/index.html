Scholarly-oriented Markdown

2015-04-12
Elegant Writing

Markdown十分轻巧，对于博客、随笔等一般要求的文章已经十分方便。然而，对于学术写作而言，则使用起来其功能未免捉襟见肘。若是使用Latex的话，自然是足够了，但是Latex写作（特别是满屏幕的反斜杠）却又不是很人道。况且有很多时候某些杂志社非要提供Word格式的稿件，即便用Pandoc将Latex文档转为Word，也还是难免需要调整半天格式。因此，出于学术写作目的的Markdown解决方案还有待观察。

Pandoc Markdown

Pandoc提供了多种格式之间相互转换的方案。此外，Pandoc还提供了多种Markdown的扩展，以形成了Pandoc Markdown。其语法的中文翻译版可参阅：http://pages.tzengyuxio.me/pandoc/。另外一份的写作说明可参阅：http://pandoc.herokuapp.com/。其中显著的特点是对参考文献以及数学公式进行了扩充，从而使之有取代Latex的可能。然而，其解决方案依然并不理想。

R Markdown

当撰写统计论文的时候，特别适合採用R Markdown。可以一键生成结果，预览公式、统计图表等。

StackEdit

当撰写工业工程或运作类论文的时候，特别适合採用线上写作工具StackEdit，可以方便地进行实时预览。

Scholdoc: A solution for Scholaryly Markdown

Scholdoc颇具雄心地提出了一套解决方案，也看起来十分具有前景。具体可参阅GitHub上的专案：https://github.com/timtylin/scholdoc。不过目前还在测试阶段，预计将在2015年秋季发布。
