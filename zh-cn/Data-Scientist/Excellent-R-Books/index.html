R语言精彩书籍

2015-08-13
Data Scientist

有关R语言的书籍实在太多太多，甚至已经到了眼花缭乱的程度了。很幸运地是，去年看到了这份R书精选16本（更新）。这份书单算是目前见到最好的一份R语言学习资料清单了。

特别好玩地是，我当年初学R时，在网上选了好久书籍。在Amazon.com上翻箱倒柜看评论，也是折腾了两三天。最后买了三本：

  • R in Action译作：R语言实战
  • The Art of R Programming译作：R语言编程艺术
  • ggplot2 Elegant Graphics for Data Analysis译作：ggplot2 数据分析与图形艺术

这三本书不必多说。第一本是非常全面的，并且在编写体例上也十分符合做数据分析的流程。第二本对R语言较为基础的一面进行了介绍，看完可以内力大增。R在绘图方面的优势，我觉得主要是ggplot2和随后的ggvis的功劳，因此第三本也是必看的。

后来学习statistical learning，便挑选了The Elements of Statistical Learning以及An Introduction to Statistical Learning。由于自己的背景更加偏重统计而非计算机科学，因此这两本书用来学习statistical learning也就再合适不过了。前者讲得非常理论，但是若是熟悉统计，定会十分喜欢；后者则更加强调R语言实现，这也是统计学者喜欢的。再加之作者更是大方地提供了这两本书的PDF版本、R代码、Slides，更是后来在推出了视频课程。在视频教学中，Hastie和Tibshirani更是请出了诸多大神进行了访谈。因此，学习machine learning的小伙伴们千万不要错过这个系列哟！

在此之前我并未看到这份去年看到了这份R书精选16本（更新）书单，但是最终的选择却是一样。所以当我后来发现这份书单时，便觉得博主大人眼界十分高明。

再往后，越来越觉得学习econometrics时，若是辅以R代码和模拟方法便能够进展迅速。于是便选了大神书单上推荐的Introduction to Scientific Programming and Simulation Using R。读完之后，也是十分惊喜。这本书在scientific calculation的部分十分精彩，包括求根、数值积分和最优化三个部分。接下来的几章略显冗长。接著的Monto-Carlo simulation的部分也是十分精彩。因此，有了本书作为根基，可以尽情享受使用simulation来学习统计理论了。

Hadley的Advanced R programming及R packages均已完成。Hadley出品，必属精品，这个也不用多说。希望接下来有空，来好好学学书单上推荐的最后两本：R Programming for Bioinformatics，以及Chambers大人的Software for Data Analysis - Programming with R。
