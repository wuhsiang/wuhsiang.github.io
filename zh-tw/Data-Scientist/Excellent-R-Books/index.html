R語言精彩書籍

2015-08-13
Data Scientist
评论

有關R語言的書籍實在太多太多，甚至已經到了眼花繚亂的程度了。很幸運地是，去年看到了這份R书精选16本（更新）。這份書單算是目前見到最好的一份R語言學習資料清單了。

特別好玩地是，我當年初學R時，在網上選了好久書籍。在Amazon.com上翻箱倒櫃看評論，也是折騰了兩三天。最後買了三本：

  • R in Action譯作：R語言實戰
  • The Art of R Programming譯作：R語言編程藝術
  • ggplot2 Elegant Graphics for Data Analysis譯作：ggplot2 數據分析與圖形藝術

這三本書不必多說。第一本是非常全面的，並且在編寫體例上也十分符合做數據分析的流程。第二本對R語言較為基礎的一面進行了介紹，看完可以內力大增。R在繪圖方面的優勢，我覺得主要是ggplot2和隨後的ggvis的功勞，因此第三本也是必看的。

後來學習statistical learning，便挑選了The Elements of Statistical Learning以及An Introduction to Statistical Learning。由於自己的背景更加偏重統計而非計算機科學，因此這兩本書用來學習statistical learning也就再合適不過了。前者講得非常理論，但是若是熟悉統計，定會十分喜歡；後者則更加強調R語言實現，這也是統計學者喜歡的。再加之作者更是大方地提供了這兩本書的PDF版本、R代碼、Slides，更是後來在推出了視頻課程。在視頻教學中，Hastie和Tibshirani更是請出了諸多大神進行了訪談。因此，學習machine learning的小夥伴們千萬不要錯過這個系列喲！

在此之前我並未看到這份去年看到了這份R书精选16本（更新）書單，但是最終的選擇卻是一樣。所以當我後來發現這份書單時，便覺得博主大人眼界十分高明。

再往後，越來越覺得學習econometrics時，若是輔以R代碼和模擬方法便能夠進展迅速。於是便選了大神書單上推薦的Introduction to Scientific Programming and Simulation Using R。讀完之後，也是十分驚喜。這本書在scientific calculation的部分十分精彩，包括求根、數值積分和最優化三個部分。接下來的幾章略顯冗長。接著的Monto-Carlo simulation的部分也是十分精彩。因此，有了本書作為根基，可以盡情享受使用simulation來學習統計理論了。

Hadley的Advanced R programming及R packages均已完成。Hadley出品，必屬精品，這個也不用多說。希望接下來有空，來好好學學書單上推薦的最後兩本：R Programming for Bioinformatics，以及Chambers大人的Software for Data Analysis - Programming with R。
