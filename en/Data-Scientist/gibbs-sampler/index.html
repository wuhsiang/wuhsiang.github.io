An example of Gibbs sampler (with R code)

2015-10-20

Algorithm

Gibbs sampler cycles through a set of conditional distributions of $$\pi$$. If we break $$\theta$$ into several “groups” or “blocks” of parameters, then the Gibbs sampler can be presented as[^rossi]:

  1. Start with initial values $$\theta_{0}$$.
  2. Sample from
    $$\begin{array}
    {lll}
    \theta{11} & \sim & f{1}(\theta{1} | \theta{02}, …,\theta{0p}) \
    \theta    {12} & \sim & f{2}(\theta{2} | \theta{11}, \theta{03}, …,\theta{0p}) \
    .. & .. & .. \
    \theta    {1p} & \sim & f{p}(\theta{p} | \theta{11}, …, \theta{1,p-1})
    \end{array}
    $$
    to obtain the first iterate.
  3. Repeat, as necessary.

An example

Consider the latent variable formulation of the binary probit model:
$$\begin{array}
{ll}
& z{i} = x{i}’ \beta + \varepsilon{i}, \ \varepsilon{i} \sim N(0,1) \
& y{i} = 0 \text{ if } z{i} < 0;\ y = 1 \text{ otherwise}.
\end{array}
$$

We use data augmentation, i.e., regarding the $$n$$-dimensional vector of $$z$$ as part of the parameters, namely, $$\theta’ = (z’, \beta)$$. Thereby, we can use the following Gibbs sampler:
$$\begin{array}
{ll}
& z | \beta, X, y \
& \beta | z, X.
\end{array}
$$

We present the Gibbs sampler for binary probit model as follows[^rossi]:

  1. Start with $$\beta_{0}$$.
  2. Draw $$z$$ from $$z | \beta, y, X$$ from $$\text{trun}{(a{i},b{i})}(x’{i} \beta{0}, 1)$$, with $$a{i} = 0$$ if $$y{i} = 1, - \infty$$ otherwise, and $$b{i} = 0$$ if $$y_{i} = 0, \infty$$ otherwise[^note].
  3. Draw $$\beta_{1}$$ from: $$\beta | z, X \sim N(\tilde{\beta}, (X’X + A)^{-1})$$ where $$\tilde{\beta} = (X’X + A)^{-1} (X’z + A \bar{\beta})$$.
  4. Repeat, as necessary.

[1] 3.76 4.57 -3.87

Since the Gibbs sampler with data augmentation always results in high autocorrelation, we may need more iterations to ensure convergence. We find that estimated parameters $$\hat{\beta} = (3.76, 4.57, -3.87)$$ is not that close to the true values $$\beta = (4, 6, -5)$$. In consequence, we need to check the performance of Gibbs sampler before reporting the results.

[^rossi]: Rossi, P. E., Allenby, G. M., & McCulloch, R. (2012). Bayesian statistics and marketing. John Wiley & Sons.
[^note]: There is an error in Rossi et al. (2012).
