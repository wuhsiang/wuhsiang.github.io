magrittr operators

magrittr provides a forward-pipe operator for R. Intorducing magrittr gives several examples to illstrate how it works. These operators include:

  • forward-pipe operator %>%
  • tree-pipe operator %T>%
  • exposition-pipe operator %$%
  • compound assignment pipe operator %<>%

%>% operator has several new features:

  1. By default the left-hand side (LHS) will be piped in as the first argument of the function appearing on the right-hand side (RHS).
  2. When the LHS is needed at a position other than the first, we can use the dot ‘.’, as placeholder.
  3. We can use the dot to create functions, which will not be confused with a placeholder when using in the same expression.
